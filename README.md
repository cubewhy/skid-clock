## 使用指南

### 菜单

在时钟页面移动摇杆或者点击返回键即可进入主菜单

任何页面长按返回键可返回到主菜单

在菜单中上下移动摇杆切换选项, 向左移动返回上级菜单, 向右/点击确认选项

### 校时

> 若条件允许可以使用联网校时 (NTP)

1. 进入菜单, 选择 Adjust Settings
2. 左右移动摇杆选择修改内容, 上下移动修改数值
3. 按下摇杆中键确认修改 / 或按返回键放弃修改

### 连接 Wi-Fi

1. 进入菜单, 选择 WiFi Settings
2. 选择 Scan Networks 来连接到无线网络 或者选择 Add Hidden WiFi 来连接隐藏 WiFi

注: 连接 Wi-Fi 后会自动请求NTP服务器校时

### 停表(秒表)

停表 (Stopwatch) 可以在 Time Tools 菜单中找到

- 按摇杆中键开始/暂停计时
- 向下拨动摇杆重置计时器

### 倒计时

倒计时 (Countdown) 可以在 Time Tools 菜单中找到

- 上下移动摇杆调整数值, 左右移动摇杆切换修改的数位
- 按摇杆中键开始/暂停倒计时
- 向下拨动摇杆重置计时器

### 番茄钟

番茄钟 (Pomodoro) 可以在 Time Tools 菜单中找到

- 按摇杆中键开始/暂停计时

### 桌宠

桌宠会在 IDLE (待机) 3分钟后自动进入

也可以从菜单手动进入 (Desktop Pet)

- 向上拨动摇杆可以移动宠物
- 扭动变阻器可以摸头

## 游戏游玩指南

### 贪吃蛇 (Snake Game)

> From [Wikipedia](https://zh.wikipedia.org/zh-cn/%E8%B4%AA%E9%A3%9F%E8%9B%87):
>
> 在游戏中，玩家操控一条细长的直线（俗称蛇或虫），它会不停前进，玩家只能操控蛇的头部朝向（上下左右），
> 一路拾起触碰到之物（或称作“豆”），并要避免触碰到自身或者其他障碍物。每次贪吃蛇吃掉一件食物，它的身体便增长一些。
> 吃掉一些食物后会使蛇的移动速度逐渐加快，让游戏的难度渐渐变大

在这个版本的贪吃蛇里, 碰到墙只会穿过去, 不会死

唯一的死亡判定条件是碰到自己的身体

- 使用摇杆移动
- 吃掉水果来增加分数/身体长度

### 五子棋 (Gomoku Game)

> From [Wikipedia](https://zh.wikipedia.org/zh-cn/%E4%BA%94%E5%AD%90%E6%A3%8B)
>
> 五子棋，是一种两人对弈的纯策略型棋类游戏，通常双方分别使用黑白两色的棋子，轮流下在棋盘直线与横线的交叉点上，先在横线、直线或斜对角线上形成5子连线者获胜。

### 2048

> From [Wikipedia](<https://zh.wikipedia.org/zh-cn/2048_(%E9%81%8A%E6%88%B2)>)
> 《2048》在4×4的网格上进行。玩家可以使用上、下、左、右四个方向键移动所有方块。但在部分情形下，不可向某些方向移动。
>  游戏开始时，网格上会出现两个数值为2或4的方块。每次移动后，另一个数值为2或4的新方块会随机出现在空方格上。
> 方块会沿着指定的方向滑动，直到被其它方块或网格边缘阻挡。如果两个相同数值的方块碰撞，它们将合并成一个方块，其数值等于两个方块的数值之和。
> 如果三个数值相同的方块碰撞，则只会合并靠近终点方向的两个方块，距起点最近的方块的数值不变。若一行或一列中的方块数值均相同，则沿着该行或该列滑动会合并前两个和后两个方块。
> 在同一移动过程中，新生成的方块不能再与其他方块合并。
> 数值较高的方块会发出柔和的光芒；但随着得分增加，光芒会不断变暗。方块数值都是2的幂，最大为131072。
> 界面右上方的记分牌会记录玩家的分数。
> 玩家的初始分数为零，每当两个方块合并时，分数会增加，得分取决于合并后方块的数值
>
> 玩家获取胜利的条件是获得数值为2048的方块。
> 达到这一目标后，玩家可以继续游戏，以获得更高的分数。
> 当玩家没有合法的移动方法时，即出现了网格上没有空方格，且相邻方块的数值均不相同的情况，游戏就直接结束。

- 用摇杆移动
